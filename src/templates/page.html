VALUE FILTERS

Separate multiple filter parameters by commas.

EVENT

Filters by the type of event. E.g. HTTP_REQUEST, CLIENT_ACCEPTED


IRULE

For iRule filter, get rid of the first slash, and then replace all remaining '/' with '-'.
E.g. if iRule is /Common/rule1 iRule filter input should be Common-rule1


FLOW

For flow filter, list the remote address first followed by a '-' and then the local address. E.g. 10.0.0.0.162:80:0-10.10.10:127:44:0


REMOTE

Remote filter format is IP Address:port number:routing domain E.g. 10.0.0.0.162:80:0


LOCAL

Local filter format is IP Address:port number:routing domain E.g. 10.10.10:127:44:0
